# Sorting-Algorithm-Visualizer-Using-PIC-Microcontroller

Following sorting algorithms are implemented in C using PIC Microntroller and displayed using OLED(SSD1306)
  1.  Selection Sort
  2.  Bubble Sort
  3.  Insertion Sort
  4.  Merge Sort
  5.  Shell Sort

Tools Used:
  1.  PIC18F4550
  2.	SSD1306 OLED (128x64)
  3.	UART
  4.	CCS C Compiler
  5.	Proteus Software

Libraries Used:
  1.	18f4550.h
  2.	SSD1306OLED.c
  3.	string.h
  4.	stdio.h
